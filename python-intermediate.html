Functions in Python

Functions are reusable blocks of code that perform specific tasks. They help make programs modular and readable. Here's how you define and use a function:

def greet(name):
    return f"Hello, {name}!"

# Calling the function
message = greet("Alice")
print(message)

Functions can also take default arguments, return multiple values, and use keyword arguments for flexibility.

Modules in Python

Modules are files containing Python definitions and statements. They allow you to organize your code into manageable pieces. Use the import statement to use a module:

# Importing a module
import math

# Using a function from the math module
result = math.sqrt(16)
print(result)  # Output: 4.0

You can also create your own modules by saving Python code in separate files.

File Handling

Python makes it easy to read and write files. Use the open() function to work with files:

# Writing to a file
with open("example.txt", "w") as file:
    file.write("Hello, Python!")

# Reading from a file
with open("example.txt", "r") as file:
    content = file.read()
    print(content)

Always use the with statement to ensure proper resource management.

Error Handling

Handle exceptions gracefully using try, except, and finally blocks:

try:
    result = 10 / 0
except ZeroDivisionError as e:
    print("Error:", e)
finally:
    print("Execution complete.")

This helps make your programs robust and user-friendly.
