Java Syntax Basics

Understand the basics of Java syntax, including the use of semicolons, comments, and the structure of a Java program. This is the foundation for writing efficient and readable Java code.

1. Comments:
In Java, comments serve the same purpose as in other languages: explaining code or making notes for better readability. Java supports:
Single-line comments: Start with //.
Multi-line comments: Enclosed within /* */.

2. Indentation:
Java encourages consistent indentation but enforces code blocks with curly braces {}. This enhances readability.
