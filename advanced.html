Object-Oriented Programming (OOP)

Object-Oriented Programming (OOP) is a paradigm that organizes code into classes and objects. It helps in designing modular and reusable code.

Core OOP Concepts:

  • Classes and Objects: A class is a blueprint, and an object is an instance of that class.
  • Encapsulation: Bundling data and methods into a single unit.
  • Inheritance: Allowing a class to derive properties and methods from another class.
  • Polymorphism: Methods in different classes can have the same name but behave differently.

Example:

class Vehicle:
    def __init__(self, name, max_speed):
        self.name = name
        self.max_speed = max_speed

    def display(self):
        print(f"Vehicle: {self.name}, Speed: {self.max_speed} km/h")

class Car(Vehicle):
    def __init__(self, name, max_speed, doors):
        super().__init__(name, max_speed)
        self.doors = doors

    def display(self):
        print(f"Car: {self.name}, Speed: {self.max_speed} km/h, Doors: {self.doors}")

car = Car("Tesla", 200, 4)
car.display()

Algorithms

Learn and implement algorithms to solve complex problems efficiently. Understanding sorting, searching, and recursion is essential.

  • Sorting: Algorithms like Quick Sort, Merge Sort, and Bubble Sort.
  • Searching: Binary Search and Linear Search.
  • Recursion: Solving problems by breaking them into smaller sub-problems.

Example: Merge Sort

def merge_sort(arr):
    if len(arr) > 1:
        mid = len(arr) // 2
        left = arr[:mid]
        right = arr[mid:]

        merge_sort(left)
        merge_sort(right)

        i = j = k = 0
        while i < len(left) and j < len(right):
            if left[i] < right[j]:
                arr[k] = left[i]
                i += 1
            else:
                arr[k] = right[j]
                j += 1
            k += 1

        while i < len(left):
            arr[k] = left[i]
            i += 1
            k += 1

        while j < len(right):
            arr[k] = right[j]
            j += 1
            k += 1
arr = [38, 27, 43, 3, 9, 82, 10]
merge_sort(arr)
print(arr)

Advanced Data Structures

Gain mastery over advanced data structures like trees, graphs, heaps, and hashmaps to handle data efficiently.

  • Binary Trees: Hierarchical structures for quick searches.
  • Graphs: Useful for network modeling.
  • Heaps: Efficient for priority queues.
  • Hashmaps: Fast key-value lookups.

Example: Graph Representation

class Graph:
    def __init__(self):
        self.graph = {}

    def add_edge(self, u, v):
        if u not in self.graph:
            self.graph[u] = []
        self.graph[u].append(v)

    def display(self):
        for node in self.graph:
            print(f"{node} -> {', '.join(map(str, self.graph[node]))}")

g = Graph()
g.add_edge(0, 1)
g.add_edge(0, 2)
g.add_edge(1, 2)
g.add_edge(2, 0)
g.add_edge(2, 3)
g.add_edge(3, 3)
g.display()
